# 版型六：曙光航纪 v1.0

> 2026-06-25 与产品方确认完毕。10 项问题全部澄清。

| 狼人 | 海妖 | `siren` | 1 |

总计 12 人，屠边。无守卫。

## 2. 全局规则

- 标准屠边，有警长，有遗言，夜死无遗言。
- 女巫每晚安只能用一瓶药（本版型确认；此规则实际应为全版型通用，后续统一修）。
- 女巫首夜不可自救。
- 预言家不可空验。

## 3. 海妖 (siren)

### 3.1 基础属性
- 阵营：狼人
- 与狼人见面，参与狼队讨论和刀人
- 每晚独立选择风向（在狼人选刀之前）

### 3.2 风向技能

| 风向 | ID | 偏移 |
|------|-----|------|
| 无风 | `calm` | 无偏移 |
| 顺风 | `tailwind` | +1（2→3，12→1 循环） |
| 逆风 | `headwind` | -1（2→1，1→12 循环） |

- 不可连续两晚相同风向
- 受风向影响的技能：狼刀、女巫毒药、预言家查验
- 女巫解药**不受**风向影响——女巫看到原始刀口，解药救偏移后的实际受伤玩家
- 海妖死亡后风向固定为最后一晚的选择，不再改变

## 4. 船长 (captain)

### 4.1 基础属性
- 阵营：好人神职
- 第二晚起，每晚**必须**选择一名**其他**存活玩家登船（不可选自己，不可空过）
- 船长死亡后不需要登船
- 海妖和船长免疫溺亡

### 4.2 登船效果 — 有风（顺风/逆风）

**登船玩家**：不受风向影响（指向 TA 的技能不被偏移；但 TA 正常承伤）

**溺亡**：登船玩家「上风向」相邻玩家溺亡
- 顺风登船 N → N-1 溺亡
- 逆风登船 N → N+1 溺亡
- 溺亡不可被女巫解药救（女巫只能看到狼刀），是必然死亡
- 死亡原因 ID：`DROWN`

### 4.3 登船效果 — 无风

**登船玩家**：
- 免疫狼刀（类似守卫守护效果）
- 不免疫毒药
- 不会被奶穿（如果将来版型有守卫，无风登船不触发同守同救死亡）
- 无溺亡

## 5. 风向偏移计算

```
applyWind(originalSeat, wind, boardedSeat):
  if wind == CALM:       return originalSeat  // 不偏移
  if originalSeat == boardedSeat: return originalSeat  // 船上免疫偏移
  if wind == TAILWIND:   return (originalSeat % 12) + 1      // +1 循环
  if wind == HEADWIND:   return ((originalSeat + 10) % 12) + 1  // -1 循环
```

## 6. 夜间流程

```
夜晚 N：
1. 海妖选择风向（N≥1；不可与昨晚相同；若海妖已死则风向固定）
2. 船长选择登船（N≥2；必须选一名其他存活玩家；船长死亡则跳过）
3. 有风时：系统计算溺亡（登船 N → 顺风 N-1 / 逆风 N+1 溺亡）
4. 狼人选刀（海妖参与；记录原始刀口）
5. 女巫行动（看到原始刀口号码；解药/毒药二选一；解药救偏移后的实际承伤玩家）
6. 预言家查验（原始号码→偏移→查验实际目标底牌）
7. 天亮：系统应用风向偏移，结算狼刀/毒药/溺亡死亡
```

## 7. 天亮宣布

每天白天发言前，法官宣布「昨夜X风」。

| 船长状态 | 行为 |
|----------|------|
| 船长存活 | 正常宣布 |
| 船长死于当晚 | 法官代为宣布该晚风向 |
| 船长死于前一个白天 | 无人宣布，玩家不知道当夜风向 |

> 风向效果仍然生效（海妖还在选），只是玩家不知道。若海妖也已死，风向固定为海妖最后一晚的选择。

## 8. 完整例子（顺风，night≥2，船长登船2号）

狼人选刀前，1号溺亡（顺风登2→溺1）。

| 原始目标 | 偏移 | 实际 | 结果 |
|----------|------|------|------|
| 刀 1 | 1→2 (1溺亡但刀仍偏移) | 2号被刀 | 2号死（在船上不免疫伤害） |
| 刀 2 | 不偏移（船） | 2号被刀 | 2号死 |
| 刀 3 | 3→4 | 4号被刀 | 4号死 |
| 验 2 | 不偏移（船） | 验2号 | 得2号底牌 |
| 毒 2 | 不偏移（船） | 毒2号 | 2号中毒 |
| 验 3 | 3→4 | 验4号 | 得4号底牌 |

## 9. 边界情况速查

| 场景 | 处理 |
|------|------|
| 海妖死亡 | 风向固定，不改变。溺亡继续触发。船长继续登船。 |
| 船长死亡（夜） | 当夜法官代为宣布风向 |
| 船长死亡（昼） | 下一夜风向未知（效果照旧），溺亡照常触发 |
| 无风 + 船长死亡 | 船长死当晚宣布「昨夜无风」，之后无人宣布 |
| 无风登船 | 船上玩家免疫狼刀（类似守卫），无奶穿风险，不免疫毒药 |
