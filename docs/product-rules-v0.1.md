# 十二点天黑产品规则规格 v0.1

## 1. 产品定位

“十二点天黑”用于线下狼人杀发牌和流程辅助。第一版重点解决固定版型发牌、玩家私密看身份、法官权限、无法官流程提示、夜间操作记录和基础结算。

第一版先将玩家手机定位为“私密看身份工具”。夜间操作可以由法官手机或公共手机集中完成。玩家个人手机夜间私密操作作为后续能力预留。

## 2. 游戏模式

### 2.1 有法官模式

- 法官创建房间，获得房间号。
- 法官选择固定版型。
- 玩家输入房间号加入房间，并绑定座位号。
- 法官确认满员后点击发牌。
- 玩家只能查看自己的身份。
- 法官可以查看所有玩家身份、当前版型、夜间行动记录和结算结果。
- 法官控制天黑、夜间行动、天亮、投票、遗言、技能发动和游戏结束。

### 2.2 无法官模式

- 系统充当流程法官。
- 玩家仍然只能查看自己的身份。
- 不存在任何玩家可查看全身份的入口。
- 夜间由公共手机按版型流程提示行动。
- 每个身份行动时只显示该身份当前需要操作的信息。
- 每个身份操作完成后立即进入遮罩或黑屏状态，避免后续身份看到前一身份信息。

## 3. 全局默认规则

- 默认 12 人局。
- 默认屠边规则。
- 默认可上警。
- 默认有遗言。
- 夜里死亡玩家没有遗言。
- 白天被放逐玩家有遗言。
- 警长死亡可以选择移交警徽或撕毁警徽。
- 狼人允许空刀。
- 预言家、通灵师不允许空验。
- 女巫首夜不可自救。
- 女巫只有一瓶解药和一瓶毒药，各只能使用一次。
- 守卫不可以连续两晚守同一名玩家。
- 白痴被放逐后出局并公布身份。
- 公开展示统一使用“白痴”作为名称；程序内部身份 ID 可继续使用 `idiot`。

## 4. 投票与 PK

- 白天投票若出现平票，进入 PK。
- PK 玩家为当前最高票平票玩家。
- PK 发言顺序按当天白天发言顺序反向处理：当天白天后发言的人，在 PK 阶段先发言。
- PK 后再次投票。
- PK 再次平票，则当天无人出局。

## 4.1 上警与退水

- 上警阶段发生在第一夜所有身份操作结束之后，第一天白天发言之前。
- 法官记录上警玩家号码后，所有玩家公开可见上警名单。
- 上警玩家可以退水。
- 已退水玩家不可参与第一轮警徽投票。
- 如果第一轮警徽投票后，仍留警上玩家出现平票 PK，则除平票玩家外所有人可以参与第二轮警徽投票。
- 第二轮警徽投票仍平票，则警徽流失。

## 5. 死亡技能限制

### 5.1 猎人

- 猎人被狼刀、被放逐时可以开枪。
- 猎人被毒杀时不能开枪。
- 猎人作为最后一神时不能开枪。

### 5.2 狼王、机械猎人、盗宝猎人

- 狼王被狼刀、被放逐时可以发动技能。
- 狼王被毒杀时不能发动技能。
- 机械狼模仿猎人时，若因死亡触发猎人技能，按机械猎人处理。
- 盗宝大师使用猎人牌时，若因死亡触发猎人技能，按盗宝猎人处理。
- 狼王、机械猎人、盗宝猎人作为狼人阵营最后一狼时不能开枪。

判断“最后一神”或“最后一狼”时，应基于场上存活玩家的当前阵营身份状态，而不是单纯基于身份名称。

## 6. 夜间保护与死亡结算

- 被摄梦玩家当夜免狼刀、免毒。
- 被守卫守护玩家当夜免狼刀、免毒。
- 女巫解药可以救被狼刀玩家，但首夜不可自救。
- 同一名玩家当夜同时被守卫守护、被女巫使用解药救起时，该玩家死亡。
- 同守同救死亡不是狼刀死亡，也不是毒杀死亡。
- 同守同救对象若为猎人、狼王，按非毒杀死亡处理，可以发动死亡技能，除非触发最后一神或最后一狼限制。
- 摄梦人连续两晚摄同一名玩家时，该玩家死亡。
- 被毒杀的猎人、狼王不能发动技能。
- 机械狼模仿守卫时，如果守到女巫毒药发动目标，女巫反伤出局。

## 7. 版型一：预女猎白混

### 7.1 身份组成

- 神职阵营：预言家、女巫、猎人、白痴
- 平民阵营：平民 x3、混血儿
- 狼人阵营：狼人 x4

### 7.2 混血儿

- 首夜选择一名玩家作为榜样。
- 胜利条件跟随榜样阵营。
- 自身属性为平民。
- 无夜间技能。
- 被预言家查验永远显示为好人。

### 7.3 白痴

- 被放逐后出局。
- 出局时公布身份。

## 8. 版型二：假面舞会

### 8.1 身份组成

- 神职阵营：预言家、女巫、舞者、白痴
- 平民阵营：平民 x4
- 狼人阵营：假面、狼人 x3

### 8.2 舞者

- 自第二夜起，每晚选择 3 名玩家共舞。
- 若 3 名玩家同阵营，无事发生。
- 若 3 名玩家不同阵营，人数少的一方死亡。
- 舞者免疫女巫的毒。
- 当舞者选择自己参与共舞时，当夜共舞的 3 人均免疫狼刀。
- 每名玩家只能参与一次共舞。

### 8.3 假面

- 假面与狼人互不见面。
- 假面每晚验证一名玩家是否在舞池中。
- 假面可以给一名玩家面具。
- 面具只影响舞池结算时该玩家被视作的阵营。
- 面具不改变真实身份、真实阵营、胜利条件，也不影响其他查验结果。
- 假面免疫女巫的毒。
- 连续两晚不能验证同一名玩家。
- 连续两晚不能给同一名玩家面具。

## 9. 版型三：盗宝大师

### 9.1 身份池

盗宝大师版型不能按普通 12 张身份牌直接相加。该版型应区分“场上玩家牌”和“盗宝牌堆”。

基础身份池：

- 神职阵营：通灵师、毒师、猎人、摄梦人、蒙面人
- 平民阵营：平民 x5
- 狼人阵营：狼王、狼人 x3
- 特殊玩家牌：盗宝大师

当前固定发牌方式：

- 场上 12 张玩家牌包含盗宝大师。
- 盗宝大师额外获得 3 张盗宝牌。
- 这 3 张盗宝牌固定为：狼人、平民、1 张神职牌。
- 场上玩家牌应从基础身份池中移除这 3 张盗宝牌后生成。
- 因此场上实际玩家牌为：盗宝大师 + 基础身份池剩余 11 张牌。

### 9.2 特殊全局规则

- 本版型首夜狼人无法发动击杀技能。

### 9.3 盗宝大师

- 首夜获得 3 张盗宝牌。
- 当前固定发给盗宝大师：狼人、平民、1 张神职牌。
- 当前神职牌池不包含蒙面人。
- 后续预留配置：允许盗宝大师拿到蒙面人。
- 当前因盗宝牌中必定包含狼人牌，盗宝大师固定为狼人阵营。
- 程序内部仍应保留“根据盗宝牌堆判断阵营”的逻辑，方便未来规则变更。
- 盗宝大师每晚选择一张盗宝牌，并使用该牌技能。
- 连续两晚不能选择同一张盗宝牌。

### 9.4 毒师

- 拥有一瓶毒药。
- 可在夜间毒杀一名玩家。
- 被毒杀的猎人或狼王无法发动技能。

### 9.5 摄梦人

- 每晚必须摄一名玩家。
- 被摄玩家当夜免狼刀、免毒。
- 连续两晚摄同一名玩家，该玩家死亡。

### 9.6 蒙面人

- 蒙面人为神职阵营。
- 夜间被毒、被摄、被狼刀时，当晚不会死亡。
- 蒙面人进入中毒或负伤状态。
- 第二天蒙面人自己发言结束后死亡。
- 当盗宝蒙面狼是场上最后一张狼人牌时，被抗推可以主动发动蒙面技能不死。
- 入夜后盗宝蒙面狼可以多一刀。
- 此时盗宝大师必须切换为蒙面人牌才可生效。

## 10. 版型四：机械狼通灵师

### 10.1 身份组成

- 神职阵营：通灵师、女巫、猎人、守卫
- 平民阵营：平民 x4
- 狼人阵营：机械狼、狼人 x3

### 10.2 通灵师

- 每晚必须查验一名存活玩家的具体身份。
- 如果查验对象是机械狼，结果显示为机械狼模仿的身份。

### 10.3 机械狼

- 机械狼夜间可模仿一名玩家。
- 机械狼不与其他三名狼人见面。
- 机械狼不可自爆。
- 其他狼人出局后，机械狼方可带刀。

### 10.4 机械狼模仿效果

- 模仿守卫：次夜起可守护一名玩家，免狼刀和毒杀；如果守到女巫毒药发动目标，女巫反伤出局。
- 模仿女巫：次夜起可毒杀一名玩家。
- 模仿猎人：出局时可带走一名玩家，但被毒杀不能发动。
- 模仿通灵师：每晚可以查验一名玩家具体身份。
- 模仿狼人：所有狼人同伴出局后，可发动一刀技能刀；技能刀无视一切保护效果。
- 模仿平民：无特殊技能。
