# 十二点天黑

狼人杀发牌/流程辅助工具。

当前优先实现为手机网页版，避免微信小程序注册、类目、审核和开发工具配置带来的额外成本。小程序版本代码暂时保留，后续如果需要可以再迁移。

第一版网页版目标是跑通线下开局的最小闭环：

- 创建房间
- 加入房间
- 绑定 1-12 号座位
- 选择固定版型
- 随机发牌
- 玩家私密查看身份
- 法官查看全局身份
- 复盘身份与关键操作

## 开发原则

第一版先做可用闭环，不追求完整自动法官结算。复杂结算允许法官手动确认，但必须保留操作记录，方便复盘。

## 预览

网页版入口：

- [web/index.html](./web/index.html)

当前网页有两种运行方式：

- 直接打开 `web/index.html`：本地演示模式，数据只保存在当前浏览器。
- 双击 `start-web-server.cmd` 后打开 `http://localhost:5173`：本地联机模式，支持同一台电脑服务下多设备加入同一个房间。

本地测试流程：

1. 双击 `start-web-server.cmd`。
2. 在电脑浏览器打开 `http://localhost:5173`。
3. 创建房间。
4. 选择版型。
5. 选择自己的座位。
6. 用其他浏览器或手机打开分享链接加入房间，或点击“补齐测试座位”。
7. 发牌。
8. 查看身份、法官总览和复盘。

手机访问时，启动窗口会显示类似 `http://192.168.x.x:5173` 的地址。手机和电脑需要在同一个 Wi-Fi 下。如果手机打不开，通常是 Windows 防火墙没有放行 Node.js。

## 从 GitHub 克隆后运行

需要 Node.js 18 或更高版本。

```bash
npm start
```

然后打开：

```text
http://localhost:5173
```

Windows 用户也可以直接双击：

```text
start-web-server.cmd
```

当前服务是本地内存房间，重启服务后房间会清空。

如果只想快速看界面，也可以直接打开 `web/index.html`：

1. 打开 `web/index.html`。
2. 创建房间。
3. 选择版型。
4. 选择自己的座位。
5. 点击“补齐测试座位”。
6. 发牌。
7. 查看身份、法官总览和复盘。

## 注册类目建议

建议注册为普通小程序，选择非游戏类目中最贴近的工具/其他类目，不建议选择“游戏”类目。

原因：

- 本项目是线下狼人杀的发牌和流程辅助工具，不是在微信内直接游玩的小游戏。
- 当前代码使用普通小程序结构：`app.json`、多页面、WXML、WXSS。
- 微信小游戏会走小游戏开发形态和审核要求，涉及游戏资质、小游戏特别规范和不同的代码组织方式。
